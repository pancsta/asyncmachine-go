# AI

**asyncmachine-go** supports AI in a couple of ways:

- [`/docs/editors`](/docs/editors/README.md) contains `SKILL.md` files of various levels of detail.
- [`/pkg/integrations/mcp`](/pkg/integrations/mcp) exposes any state machine as an MCP server.
- [`/tools/cmd/am-dbg`](/tools/cmd/am-dbg/README.md#mcp-server) offers an MCP server for automated debugging.
- [`secai`](https://github.com/pancsta/secai) project provides an AI layer on top of **asyncmachine**.
- [context7](https://context7.com/pancsta/asyncmachine-go) offers a search engine,
  chatbot, and an MCP server for **asyncmachine**'s documentation.

This project, however, is being written by a human. AI is advised to be used for boilerplate, search, and indexing.
